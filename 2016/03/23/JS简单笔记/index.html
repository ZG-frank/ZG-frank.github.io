JS简单笔记

Frank's Blog
2016-03-23
JS
  1. JavaScript严格区分大小写，如果弄错了大小写，程序将报错或者运行不正常
  2. 数据类型：数字Number(不区分整数和浮点数；NaN无法计算结果、Infinity无限大)、字符串、布尔值、数组、对象、null、undefined
  3. 比较运算符：==比较和===比较
    ==比较会自动转换数据类型再比较，很多时候，会得到非常诡异的结果；
    ===比较不会自动转换数据类型，如果数据类型不一致，返回false，如果一致，再比较。
    由于JavaScript这个设计缺陷，不要使用==比较，始终坚持使用===比较。
    另一个例外是NaN这个特殊的Number与所有其他值都不相等，包括它自己和自己比较。可以用函数isNaN(对象)；

  4. JS有一个关键字arguments，它只在函数内部起作用，并且永远指向当前函数的调用者传入的所有参数。它不是一个Array，可利用它获得传入的所有参数。最常用于判断传入参数的个数。

  5. 不在任何函数内定义的变量就具有全局作用域。实际上，JavaScript默认有一个全局对象window，全局作用域的变量实际上被绑定到window的一个属性。所以以变量方式var foo = function () {}定义的函数实际上也是一个全局变量，因此，顶层函数的定义也被视为一个全局变量，并绑定到window对象。

  6. 由于JavaScript的变量作用域实际上是函数内部，我们在for循环等语句块中是无法定义具有局部作用域的变量的，在ES6中引入了let，可以代替var申明一个块级作用域变量。

  7. 通常用全大写来表示一个变量是常量。ES6标准引入了新的关键字const来定义常量，const与let都具有块级作用域。

  8. 在一个对象中绑定函数，称为这个对象的方法。如xiaoming里的age()方法，绑定后可以使用this关键字。要保证this指向正确，必须用obj.xxx()的形式调用！单独调用函数时this指向全局对象，也就是window。将var fn = obj.xxx()，调用fn()也是不行的。而且strict模式下函数的this指向undefined。

  9. this在JavaScript中主要由以下五种使用场景。
    1、在全局函数调用中，this绑定全局对象，浏览器环境全局对象为window 。
    2、作为对象方法使用，this绑定到该对象。
    3、在对象方法的内部的函数中，this也绑定全局对象。准确来说应该绑定到其外层函数对应的对象上，这是JavaScript的缺陷。解决方法：用一个变量在方法内部一开始就捕获this。
    4、在构造函数中，this 绑定到新创建的对象。
    5、使用 apply或call调用函数，this将会被显式设置为传入的的第一个参数。

  10. 虽然在一个独立的函数调用中，根据是否是strict模式，this指向undefined或window，不过，我们还是可以控制this的指向的！要指定函数的this指向哪个对象，可以用函数本身的apply方法，它接收两个参数，第一个参数就是需要绑定的this变量，第二个参数是Array，表示函数本身的参数。

  11. 与apply()类似的方法是call()，唯一区别是：apply()把参数打包成Array再传入；call()把参数按顺序传入。对普通函数调用，通常把this绑定为null。

  13. JavaScript的函数其实都指向某个变量。既然变量可以指向函数，函数的参数能接收变量，那么一个函数就可以接收另一个函数作为参数，这种函数就称之为高阶函数。

  14. map()方法定义在JavaScript的Array中，返回一个由原数组中的每个元素调用一个指定方法后的返回值组成的新数组。

  18. 利用map()把字符串变成整数；由于map()接收的回调函数可以有3个参数：callback(currentValue, index, array)，通常我们仅需要第一个参数，而忽略了传入的后面两个参数。不幸的是，parseInt(string, radix)没有忽略第二个参数，导致实际执行的函数分别是：

    parseInt(‘1’, 1); // NaN, 没有一进制

  20. Array的sort()方法默认把所有元素先转换为String再排序。默认情况下，对字符串排序，是按照ASCII的大小比较的。sort()也是一个高阶函数，它还可以接收一个比较函数来实现自定义的排序。return -1的结果放在前面。注意sort()方法返回的还是原来的Array。

  21. 闭包：高阶函数除了可以接受函数作为参数外，还可以把函数作为结果值返回。
