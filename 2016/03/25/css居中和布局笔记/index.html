CSS

水平居中

  1. 行内元素，设置text-align: center属性

  2. 定宽块级元素，设置左右margin值为auto

  3. 不定宽块级元素，绝对定位加transform

  4. flex

垂直居中

  1. 单行文本，设置height与line-height相等 或 padding-top = padding-bottom
  2. 多行文本，设置padding-top = padding-bottom

  3. 定高块级元素

  4. 不定高块级元素，绝对定位加transform或

  5. flex
